# Demo App: HTML Mockups at /demo/

## Overview

Добавить минимальное Django-приложение `demo`, которое раздаёт три HTML-мокапа главной страницы
по адресам `/demo/home-multiple/`, `/demo/home-offseason/`, `/demo/home-single/`.
Не связано с остальным сайтом — без авторизации, без навигации, только для демонстрации дизайна.

## Context (from discovery)

- Файлы-мокапы: `scratch/demo/Home - несколько гонок.html`, `Home - межсезонье.html`, `Home - 1 активная гонка.html`
- Существующие app-паттерны: `src/donate/` (минимальный app — `apps.py`, `urls.py`, `views.py`)
- `INSTALLED_APPS` в `src/config/settings.py:71`
- URL-роутинг в `src/config/urls.py`
- Шаблонизатор: Django templates; HTML-файлы без `{%`-тегов → копируются как есть

## Development Approach

- **testing approach**: Regular (код → тесты)
- Задачи выполняются последовательно; тесты пишутся по завершении каждой

## Solution Overview

Создаём `src/demo/` с `apps.py`, `urls.py`, `templates/demo/` (без моделей и views —
используем `TemplateView`). HTML-файлы копируем из `scratch/demo/` с человекочитаемыми именами.
Регистрируем в `INSTALLED_APPS` и подключаем маршрут `demo/`.
